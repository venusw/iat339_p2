Sugar Craft Style Guide

A template for Sugar Craft branding.

Logos

Default Nav Bar Style

CSS selector:

'a.nav-bar'

Sample code:

<a class="nav-bar" href="#"> <nav> <a href="#products">PRODUCTS</a> <a href="#contact">CONTACT</a> </nav>

Rendered element:

PRODUCTS CONTACT

H1 Text

CSS selector:

'a.h1'

Sample code:

<a href="#" class="h1">Default H1 Text</a>

Rendered element:

Default H1 Text

Default Button Style

CSS selector:

'a.button'

Sample code:

<a href="#" class="button">Default Button</a>

Rendered element:

DEFAULT BUTTON
